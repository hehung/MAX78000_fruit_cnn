# MAX78000 水果识别代码工程

# 描述
实现了摄像头实时采集图像，并将图片传递给CNN计算，从计算结果中找出与采集照片中相似度最高的水果种类并通过LCD显示出来。  
能够识别的水果包括：   
```
苹果
香蕉
葡萄簇（只针对一串葡萄，不针对单个葡萄）
猕猴桃
芒果
橘子
梨子
菠萝
石榴
草莓
西瓜
```

# 注意事项
该工程只支持TFTLCD，在进行水果识别功能验证之前需要先连接TFTLCD。   
TFTLCD大小为320X240。   

# 操作流程
1. 上电，按下SW1开始进行采集；
2. 摄像头采集水果图片，TFTLCD实时显示图片以及识别结果；
3. 按下SW1暂停程序，程序暂停后再次按下SW1启动程序。
